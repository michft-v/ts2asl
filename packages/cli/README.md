## `ts2asl compile FILE`

Compile TypeScript to AWS ASL

```
USAGE
  $ ts2asl compile [FILE]

ARGUMENTS
  FILE  input file

DESCRIPTION
  Compile TypeScript to AWS ASL
```

_See code: [dist/commands/compile/index.ts](https://github.com/Stedi/ts2asl/blob/v0.0.0/dist/commands/compile/index.ts)_
